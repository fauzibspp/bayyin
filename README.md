# Bayyin

Bayyin is a lightweight PHP native MVC framework with:
- MVC core
- CLI scaffolding
- migrations and seeders
- web CRUD generation
- API CRUD generation
- JWT authentication for API
- DataTables-ready admin modules
- soft delete, trash/restore, export CSV, and audit logging support

## Requirements
- PHP 8.1+
- Composer
- MySQL/MariaDB
